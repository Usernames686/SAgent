## 第一章：测试知识点详情

### 1.1 知识点定义

**名称**：React 中的毛玻璃效果（Glassmorphism）

**所属模块**：Vibe Coding - 提示词工程阶段

**难度等级**：D3（中等）

**知识描述**：
毛玻璃效果（Glassmorphism）是一种现代 UI 设计风格，通过半透明背景、模糊效果和微妙的边框创造出类似磨砂玻璃的视觉效果。在 React 中，主要通过 CSS 的 `backdrop-filter` 属性实现。

**核心技术**：
- CSS `backdrop-filter: blur()`
- 半透明背景 `background: rgba(255, 255, 255, 0.1)`
- 细微边框 `border: 1px solid rgba(255, 255, 255, 0.2)`

**学习目标**：
1. 理解毛玻璃效果的原理
2. 能在 React 中实现毛玻璃效果
3. 能调整参数优化视觉效果

### 4.3 A/B 测试详情

**实验设计**：
- 对照组：策略 B（案例对比法）
- 实验组：策略 C（苏格拉底式引导）
- 样本量：每组 5 名学生
- 流量分配：50% / 50%
- 实验周期：1 周

**实验结果**：

| 指标 | 对照组（策略B） | 实验组（策略C） | 差异 |
|------|-----------------|-----------------|------|
| 平均分 | 52.0 | 84.6 | +62.7% |
| 通过率 | 40% | 100% | +150% |
| 平均用时 | 10.5分钟 | 9.5分钟 | -9.5% |

**统计分析**：
- P 值：0.028（< 0.05，显著）
- 置信度：97.2%
- Cohen's d：0.85（大效应）

**结论**：苏格拉底式引导策略显著优于案例对比法，建议全量采用。
