13. 安全设计

13.1 代码沙箱安全架构

多层沙箱隔离 关键安全
隔离层技术防护范围
L1 容器隔离Docker + gVisor (runsc)系统调用过滤、内核隔离
L2 资源限制cgroups + ulimitCPU/内存/磁盘/网络限制
L3 网络隔离NetworkPolicy + 无外网禁止外联、仅内部通信
L4 文件系统只读根文件系统 + tmpfs禁止持久化写入
L5 进程限制seccomp + AppArmor限制系统调用、禁止 fork bomb

13.2 安全防护体系

安全域威胁防护措施
认证授权未授权访问、Token 泄露JWT + Refresh Token、RBAC、OAuth 2.0、IP 白名单
数据传输中间人攻击、窃听全站 HTTPS/TLS 1.3、HSTS、CSP
数据存储数据泄露、拖库bcrypt 密码哈希、AES-256 加密敏感字段、数据库加密
代码执行沙箱逃逸、恶意代码gVisor 多层隔离、资源限制、网络隔离、代码静态分析
AI 安全Prompt 注入、幻觉代码输入过滤、输出审查、幻觉检测、人工审核
API 安全DDoS、暴力破解限流（令牌桶）、WAF、验证码、IP 黑名单
隐私合规隐私泄露、违规采集数据脱敏、K-匿名化、差分隐私、GDPR/个保法合规
