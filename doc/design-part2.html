9. 接口定义

9.1 核心 API 列表

9.1.1 认证接口

方法路径描述认证请求体响应
POST/api/v1/auth/register用户注册无{email, password, nickname}{user, accessToken, refreshToken}
POST/api/v1/auth/login用户登录无{email, password}{user, accessToken, refreshToken}
POST/api/v1/auth/refresh刷新 TokenRefreshToken{refreshToken}{accessToken, refreshToken}
POST/api/v1/auth/oauth/githubGitHub OAuth无{code}{user, accessToken, refreshToken}
POST/api/v1/auth/logout退出登录JWT-{success}

9.1.2 用户接口

方法路径描述认证请求体响应
GET/api/v1/users/me获取当前用户JWT-{user, profile, stats}
PUT/api/v1/users/me更新用户信息JWT{nickname, avatarUrl, preferences}{user}
GET/api/v1/users/me/profile获取用户画像JWT-{profile}
POST/api/v1/users/me/assessment提交能力诊断JWT{answers[]}{profile, suggestedPath}
GET/api/v1/users/me/dashboard学习仪表盘JWT-{stats, path, recommendations}

9.1.3 学习路径接口

方法路径描述认证请求体响应
GET/api/v1/learning-paths/me我的学习路径JWT-{paths[]}
POST/api/v1/learning-paths/generate生成学习路径JWT{goal, timeline, commitment}{path}
PUT/api/v1/learning-paths/{id}/adjust调整学习路径JWT{reason, preferences}{path}
GET/api/v1/learning-paths/{id}/progress路径进度JWT-{progress, knowledgeState}

9.1.4 练习接口

方法路径描述认证请求体响应
GET/api/v1/exercises练习列表JWT?page&limit&difficulty&module&language{exercises[], total}
GET/api/v1/exercises/{id}练习详情JWT-{exercise, template, hints}
POST/api/v1/exercises/{id}/submit提交代码JWT{code, language}{submission, testResults}
GET/api/v1/exercises/{id}/submissions提交历史JWT?page&limit{submissions[]}
POST/api/v1/exercises/{id}/run运行代码JWT{code, language, input}{output, error}

9.1.5 AI Agent 接口

方法路径描述认证请求体响应
WS/api/v1/chatAI 对话（流式）JWT{message, context}SSE stream
POST/api/v1/agent/evaluate代码评估JWT{code, exerciseId}{evaluation, suggestions}
POST/api/v1/agent/debug调试辅助JWT{code, error, logs}{diagnosis, fix}
POST/api/v1/agent/review代码审查JWT{code, context}{review, issues}
POST/api/v1/agent/vibe氛围编程生成JWT{vibe, requirements, constraints}{code, matchScore}
POST/api/v1/agent/feedback提交 AI 反馈JWT{sessionId, rating, comment}{success}

9.2 统一响应格式

// 成功响应
{
  "code": 0,
  "message": "success",
  "data": { ... },
  "meta": {
    "requestId": "req-uuid",
    "timestamp": "2026-06-10T10:00:00Z",
    "page": 1,
    "pageSize": 20,
    "total": 100
  }
}

// 错误响应
{
  "code": 1001,
  "message": "Token 已过期",
  "errors": [
    { "field": "token", "message": "Access token expired at ..." }
  ],
  "meta": { "requestId": "req-uuid", "timestamp": "..." }
}

9.3 WebSocket 消息格式

// 客户端发送
{
  "type": "message",
  "data": {
    "content": "帮我解释一下 JavaScript 闭包",
    "context": {
      "currentCode": "function counter() { ... }",
      "exerciseId": "ex-js-023"
    }
  }
}

// 服务端流式返回
{
  "type": "stream_start",
  "data": { "sessionId": "sess-uuid", "agent": "tutor" }
}
{
  "type": "stream_chunk",
  "data": { "content": "闭包是指...", "index": 0 }
}
{
  "type": "stream_chunk",
  "data": { "content": "一个函数可以记住...", "index": 1 }
}
{
  "type": "stream_end",
  "data": { "sessionId": "sess-uuid", "tokens": 156 }
}

// 反馈
{
  "type": "feedback",
  "data": { "sessionId": "sess-uuid", "rating": 5, "comment": "解释很清楚" }
}
