# sAgent 项目指令

这是一个 **自我进化多智能体个性化学习氛围编程平台**，使用 pnpm monorepo 管理。

## 技术栈

- **后端**: NestJS 10 + TypeORM + SQLite/PostgreSQL + Socket.IO + JWT + OpenAI SDK
- **前端**: Next.js 15 + React 19 + Tailwind CSS 3 + Zustand + React Query + Monaco Editor
- **共享包**: `@sagent/shared` — 类型定义与工具函数
- **构建**: pnpm workspaces + TypeScript 5.5 (bundler mode)
- **测试**: Vitest（后端），测试文件与源码同级放置 `*.spec.ts`

## 常用命令

| 命令 | 说明 |
|---|---|
| `pnpm dev` | 同时启动前端和后端开发服务器 |
| `pnpm dev:web` | 仅启动 Web 前端 (Next.js) |
| `pnpm dev:api` | 仅启动 API 后端 (NestJS watch mode) |
| `pnpm build` | 构建所有包 |
| `pnpm test` | 运行所有包测试 |
| `pnpm typecheck` | 全项目 TypeScript 类型检查 |
| `pnpm lint` | ESLint 检查 |

## 开发约定

- 后端使用 NestJS 模块化架构，按模块组织 controller/service/module
- 智能体统一放在 `modules/agent/agents/` 下，继承基础 Agent 类
- API 接口使用 Swagger 注解 (`@nestjs/swagger`)
- WebSocket 通过 `Socket.IO` Gateway 实现
- 前端使用 App Router，页面放在 `app/dashboard/` 下
- 状态管理使用 Zustand stores，API 请求通过 React Query
- 组件使用 Tailwind CSS 类名，不写内联 style
- 编辑后先跑 `pnpm typecheck` 再提交

格式与现有文档保持一致，不要删除历史记录。
